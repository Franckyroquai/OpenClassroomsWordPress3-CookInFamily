Administration est un plugin pour le cours WordPress d’OpenClassrooms.

== Description ==

Le plugin Administration proposé par OpenClassrooms permet d’activer une page d’administration ; 
celle-ci comporte un champ pour modifier la couleur de fond d’un site WordPress.
